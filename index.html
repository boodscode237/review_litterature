Disambiguating Geographic Names in a Historical Digital Library

David A. Smith, Gregory Crane (2001)
Full Paper

Introduction

Objective: Enhance historical text understanding through toponym disambiguation. Combines text analysis with gazetteer resources to create georeferenced digital libraries.

Methodology

Uses Perseus system with 1M+ place names. Combines document context with spatial reasoning (e.g., "Lancaster" near "Philadelphia" implies Pennsylvania).

Key Findings

Achieved high recall (84%) but lower precision due to database size. Performs best on ancient texts, struggles with modern ambiguous locations.

Grounding Spatial Named Entities for Information Extraction and Question Answering

Alex Klementiev, Dan Roth, Kevin Small (2004)
Full Paper

Core Concept

Develops spatial minimality principle for disambiguation. Uses UN-LOCODE gazetteer (36k+ locations) with visualization techniques.

Innovation

First system to automatically generate geographic maps from news articles. Achieved 85.6% precision in location grounding.

Contribution

Introduces ISO-standard spatial annotation language. Achieved 93% disambiguation accuracy using IGDB gazetteer with 1.4k+ matches.

Application

Used in disease tracking (ProMED) and immigration reports. F-measure: 85 (specific locations), 72 (general terms).
